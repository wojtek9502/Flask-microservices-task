Startowanie Brokera RabbitQM na windows
- w konsoli uruchomionej jako admin
- przejść do folderu gdzie jest zainstalowany RabbitMQ i do folderu sbin:
```shell
  cd C:\Program Files\RabbitMQ Server\rabbitmq_server-3.9.4\sbin
```
- uruchomić broker poleceniem
```shell
rabbitmq-server.bat
```
- zatrzymywanie brokera poleceniem
```sh
rabbitmq-service.bat stop
```

Rabbitmq na docker
docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.9-management


mikroserwis pociąg
 Najpierw wejsc w venv: source venv/bin/actiavte na linuxie
 celery -A train_microservice.celery_runner worker  --loglevel=INFO
 celery multi stop -A train_microservice.celery_runner worker  --loglevel=INFO

celery -A train_microservice.celery_runner beat   --loglevel=INFO

na linuxie mozna odpalic worker celery razem z beat
celery -A train_microservice.celery_runner worker -n "train" -Q "celery_periodic" --loglevel=INFO
celery -A train_microservice.celery_runner beat   --loglevel=INFO
